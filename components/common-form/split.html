# Split 表单分段

开发者将 fields 参数中 type 属性设置为 split 表示在此处将表单分段。一个 field 配置仅能表示为表单元素或表单分段。

属性

属性名 属性值说明 默认值
type [String] split 无
title [String] 分段标题名 无
border [Boolean] 是否显示下分割线 false
className [String] 为分割线追加样式名 无
style [String] 自定义分割线样式 无

示例

const formFields = [
  {
    type: 'split',
    title: '客户信息'
  }
]
